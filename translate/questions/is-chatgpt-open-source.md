# Is ChatGPT open source AI?

**Answer:** No, ChatGPT is not open source. This can be somewhat confusing as the organization behind ChatGPT is called OpenAI, but ChatGPT is not published under an open license and ChatGPT does not allow you to view or modify the source code as it is not publicly available.
