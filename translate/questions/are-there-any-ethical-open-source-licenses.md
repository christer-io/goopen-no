# Are there any "ethical" open source licenses?

**Answer:** No, there are no licenses covering ethical issues in the list of licenses that are approved by the Open Source Initiative. 

There are a few licenses in early stages development developing ethical licenses. The organisation Ethical Source has a list of these: https://ethicalsource.dev/licenses/
