# Hvorfor bør jeg bruke den nyeste versjonen av Creative Commons-lisensene?

**Svar:**

Den siste versjonen av Creative Commons-lisensene er versjon 4.0. Du bør alltid bruke den nyeste versjonen av Creative Commons-lisensene for å dra nytte av de mange forbedringene som er beskrevet på lisensversjonssiden. Spesielt er 4.0 ment å være bedre egnet til internasjonal bruk, og bruk i mange ulike sammenhenger, inkludert deling av data.
