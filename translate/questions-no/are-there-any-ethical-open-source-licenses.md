# Finnes det noen "etiske" åpen kildekode-lisenser?

**Svar:** Nei, det er ingen lisenser som dekker etiske spørsmål i listen over lisenser som er godkjent av Open Source Initiative. 

Det er noen få lisenser i tidlige utviklingsstadier for å utvikle etiske lisenser. Organisasjonen Ethical Source har en liste over disse: https://ethicalsource.dev/licenses/
