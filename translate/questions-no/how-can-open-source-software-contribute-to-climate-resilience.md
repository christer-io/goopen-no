# Hvordan kan åpen kildekode-programvare bidra til klimaresiliens?

**Svar:** Programvare med åpen kildekode muliggjør utvikling av tilpasningsdyktige verktøy for å overvåke og forutsi klimapåvirkninger. Samfunn kan tilpasse disse verktøyene for å møte lokale behov, og gi dem mulighet til å bygge motstandskraft mot klimarelaterte risikoer som ekstreme værhendelser og havnivåstigning.
