# Hvordan vet jeg om en pedagogisk ressurs er en OER?

**Svar:** Det viktigste kjennetegn ved OER er dens immaterielle rettighetslisens og frihetene lisensen gir andre til å dele og tilpasse den. Hvis en leksjonsplan eller aktivitet ikke er tydelig merket eller merket som værende i det offentlige domene eller har en åpen lisens, er det ikke OER. Så enkelt er det. Mens egendefinerte opphavsrettslisenser kan utvikles for å lette utviklingen og bruken av OER, kan det ofte være enklere å bruke fritt tilgjengelige standardiserte lisenser utviklet spesielt for dette formålet, for eksempel de som er utviklet av Creative Commons eller – for programvare – de som er godkjent av Open Source Initiative.
