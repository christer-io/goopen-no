# Hvordan bør jeg bestemme hvilken lisens jeg skal velge?

Hvis du er usikker på hvilken lisens som passer best for dine behov, er det nok av ressurser for å hjelpe rettighetshavere med å velge riktig CC-lisens. CC Australia har utviklet et flytskjema som kan være nyttig for å hjelpe deg med å finne riktig lisens for arbeidet ditt. Du kan også lese casestudier av andre som bruker CC-lisenser. CC-fellesskapet kan også svare på spørsmål, og kan allerede ha tatt opp problemer du tar opp. CC-fellesskapets e-postdiskusjonslister og diskusjonsarkiver kan være nyttige ressurser.

Til slutt kan det også være lurt å konsultere en advokat for å få råd om den beste lisensen for dine behov.
