# Jeg utvikler et nettapp-prosjekt. Hvilke standarder bør jeg se på?

**Svar:** Det avhenger av hva du utvikler og hvordan produktet er ment å brukes, men det er noen generelle standarder som de fleste prosjekter bør være klar over. 

## Datasett

JSON (JavaScript Object Notation)

XML

## Tegnkoding og språk

UTF-8, ISO/IEC 10646

ISO 639-1/639-2/639-3

## Webstandarder

HTML

CSS

SSL

HTTP

XHTML (eXtensible HTML v1.0)

FTP

DNS

SMTP

## Tilgjengelighet

WCAG 2.0/2.1 (retningslinjer for tilgjengelighet for nettinnhold)

## Link og referanse

Uniform Resource Identifier (URI)

## Dokumentformater

OOXML (Office Open XML)

ODF (Open Document Format)

PDF (PDF/A-1)

EPUB 3.0/3.2 (for e-bøker)

## Multimedia

SVG (Scalable Vector Graphics)

PNG (Portable Network Graphics)

JPEG (Joint Photographic Experts Group)

Ogg MP3 (Moving Picture Experts Group: Audio Layer III)

FLAC (Free Lossless Audio Codac)

H.264 (H.264/MPEG-4 AVC)

AAC (avansert lydkoding)

MP4 (MPEG-4 del 14)
