# Hvordan fungerer lisensiering for åpen maskinvare?

**Svar:** Åpne maskinvareprosjekter bruker ofte lisenser som spesifiserer vilkårene for hvordan maskinvaredesignene kan brukes og deles. Populære åpne maskinvarelisenser inkluderer Creative Commons-lisensene og TAPR Open Hardware License. Disse lisensene kan kreve attribusjon, forby kommersiell bruk eller tillate avledede verk, avhengig av den spesifikke lisensen som er valgt.
