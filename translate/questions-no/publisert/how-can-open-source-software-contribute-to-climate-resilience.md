# Hvordan kan åpen kildekode bidra i kampen for å redde klima?

**Svar:** Programvare med åpen kildekode muliggjør utvikling av tilpasningsdyktige verktøy for å overvåke og forutsi klimapåvirkninger. 

Samfunn kan tilpasse disse verktøyene for å møte lokale behov, og gi dem mulighet til å bygge motstandskraft mot klimarelaterte risikoer som ekstreme værhendelser og havnivåstigning.

Data om hav, skog, natur og lokale forhold gir mer kunnskap. Kunnskap gir grunnlag for å ta riktige beslutninger.
