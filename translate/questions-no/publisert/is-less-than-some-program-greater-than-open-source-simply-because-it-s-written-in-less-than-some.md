# Er et program automatisk Åpen kildekode ganske enkelt fordi det er skrevet på en plattform som er åpen kildekode?

**Svar:**

Nei. Mens språk som PHP, Perl eller Python har implementeringer som er lisensiert under åpen kildekode-lisenser, gjør det ikke all kode skrevet på disse språkene eller kjøres under slike implementeringer til åpen kildekode. 

Koden som er skrevet på slike språk eller kjøres under slike implementeringer, må være lisensiert under en godkjent åpen kildekode-lisens for å være åpen kildekode.
