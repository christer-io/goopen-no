## Finnes det en definisjon på hva fri programvare er?

**Svar:** Ja. Free Software Foundation beskriver fri programvare gjennom fire grunnleggende friheter, mens Open Source Initiative bruker Open Source Definition med ti kriterier.

Definisjonene har ulikt språk og ulik historisk bakgrunn, men overlapper i stor grad i praksis. Begge peker mot programvare som kan brukes, studeres, endres og deles.

- [Free Software Foundation: Free Software Definition](https://www.gnu.org/philosophy/free-sw.html)
- [Open Source Initiative: Open Source Definition](https://opensource.org/osd)
